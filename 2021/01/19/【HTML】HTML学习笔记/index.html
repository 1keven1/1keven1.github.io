【HTML】HTML学习笔记

发表于2021-01-19|更新于2021-02-06|技术笔记学习笔记HTML
|字数总计:421|阅读时长:1分钟|阅读量:|评论数:

持续更新……

小写 记得结束标签

注释：< !– 注释 – >

Head元素：

  • title 标题
  • base 默认根url
  • meta 描述（关键词，作者，字符集）
  • style 内部样式表
  • link 定义资源引用地址（链接外部样式表）

Body元素：

  • html HTML元素

  • h1-h6 标题

  • p 段落

  • a 链接（< a href=”https://www.runoob.com">这是一个链接< /a>）

  • img 图片（< img loading=”lazy” border = “1” src=”/images/logo.png” width=”258” height=”39” />）

    • src 源位置
    • alt 图片的文字信息，在无法载入图像时替换显示
    • height width 默认单位像素

  • br 换行 写成< br/ >结束

  • hr 水平线

  • map 图像地图

  • table 表格（border=“1” cellpadding cellspacing）

    • caption 标题
    • tr 行
    • th 表头（colspan rowspan）
    • td 列
    • thead tbody tfoot 不会影响排布，可以用css设置格式

  • ul 无序列表

  • ol 有序列表（type: “A” “a” “I” “i” start）

    • li

  • dl 自定义列表

    • dt 自定义项目
    • dd 自定义描述

文本格式化

建议用strong代替b，用em代替i

  • b 粗体
  • em 着重文字
  • i 斜体文字
  • small 小号字
  • strong 加重语气
  • sub 下标字
  • ins 插入字（下划线）
  • del 删除字

链接：

< a href=”url” >这是一个链接< /a >

  • href
  • target _blank是打开新的网页
  • 图片链接 < a href=”url”>< img border src alt width height>< /a>
  • id = “tips” href = “#tips” 网页内跳转

分组：

  • div 块级
  • span 行内元素

内联样式：style

  • color
  • background-color
  • font-family
  • font-size （px）
  • text-align
  • float 图片、div悬浮
  • list-style-type 列表样式

表单和输入: form

  • 文本域：input type=”text”
  • 密码字段：input type=”password”
  • 单选按钮：input type=”radio”
  • 复选框：input type=”checkbox”
  • 提交按钮：input type=”submit”
