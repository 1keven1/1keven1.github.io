【微信小游戏】方圆——文物组装拼图小游戏

发表于2020-07-03|更新于2024-09-29|作品集游戏
|字数总计:902|阅读时长:2分钟|阅读量:|评论数:

游戏截图

游戏介绍

扮演制作文物的匠人，在或方或圆的文物模块中进行有序地拼合，领略从古至今人民群众创造的智慧与隐藏于文物背后的历史故事。

基本信息

  • 平台：Android、IOS
  • 类型：益智、拼图

玩法简介

  • （基础玩法）
    依据提示，按一定顺序将拼图块拖动至相应的位置，完成对该文物的拼合铸造。
  • （一定顺序）
    通常依据该文物的实际创造过程而定，如从下到上；从里到外等。
  • （提示系统）
    游戏过程中可以随时点击关卡上方的卷轴按钮打开提示卷轴，查看以文字形式呈 现的拼合顺序。
  • （帮助系统）
    每五次拼合失败获得一次使用帮助的机会，点击对应按键，系统会自动将部件拼合到指定位置。

设计思路

  • 主题：国博；创造

  • 核心概念：合理、有序的创造过程“随方逐圆”一词，具有“根据物体的形状、地形的高低等做出与之相适应的设计”这样的含义。古时匠人依据实际需求逐步地将文物创造而出的过程，正如人民群众脚踏实地创造历史这一漫长进程的缩影。

我们在机制设计上，尝试以拼图的形式还原文物创造的过程，并提取出创造顺序中的“合理”“有序”这两个特点，希望玩家在游戏过程中也能领略一番文物设计的精巧之处，感受传承千年的创造之美。

分工

组内有四名同学：我、庭夏、九叶、柠檬茶，分工如下：

  • 我：微信可视化程序、玩法设计；
  • 庭夏：策划、推广；
  • 九叶：策划、音效、音乐；
  • 柠檬茶：美术；

技术路线

  • 对每个拼图进行编号，标明等级；

  • 使用CSV表格储存每一关的对应编号拼图的位置；

  • 当玩家放手的时候，先检查拼图等级。如果可以拼，再检查此拼图距对应位置的距离，如果小于一个值则平滑移动过去，并发送成功信号。

  • 每当接收到成功信号，检测当前等级是否拼完，拼完则可以拼下一个等级的部件。

其他

  • 部分音效来自爱给网CC协议曲库
  • BGM为《倾杯乐》（片段）
  • 微信小游戏制作工具为可视化编程，虽然功能有限而且有点小bug，但是一直在更新，而且上手简单无比。如果有想自己制作小游戏的，可以点击这个链接，放心大胆地去尝试一下。

收获

因为是用的微信可视化工具，所以说功能上受限还是很大的，还有各种蜜汁bug。不过恰好锻炼了我的绿皮能力（不是）。使用最简单的工具想尽一切方法，尽可能达到设计的效果

不足

对于等级的逻辑还是有问题，没有采用每一块拼图检测自己的父元素，而是使用了一个关卡统一一级元素拼装完毕才能拼接二级元素。这导致了即使这个元素的父元素已经安装完毕，只要关卡中别的组件的父元素没有安装完毕就无法安装，逻辑上不是很正确，但目前没有想到很好的解决方法。
